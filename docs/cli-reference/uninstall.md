## Overview

Removes a previously installed addon component from the cluster.

```
podplane uninstall <component> [flags]
```

## Options

| Flag | Description |
| --- | --- |
| `-f, --file string` | Path to the cluster config file (default: `podplane.cluster.jsonc` in the current directory) |
